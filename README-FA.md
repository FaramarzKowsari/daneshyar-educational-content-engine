# اصلاح کارت‌های قابلیت دانشیار

علت مشکل:
کارت‌های بخش قابلیت‌ها فقط معرفی بصری بودند و هیچ لینک یا Event Listener نداشتند.

اصلاحات:
- هر شش کارت اکنون قابل کلیک است.
- گفت‌وگو مستقیماً پنل Chat را باز می‌کند.
- خلاصه، آزمون، فلش‌کارت، نقشهٔ مفهومی و اسلاید پنل تولید محتوا را باز و ابزار مربوط را برجسته می‌کنند.
- نقشهٔ مفهومی که در Backend وجود داشت ولی در رابط جا افتاده بود، به پنل تولید محتوا اضافه شد.
- بدون PDF آماده، پیام راهنما نمایش داده می‌شود.
- تولید خودکار با کلیک روی کارت انجام نمی‌شود تا مصرف یا هزینهٔ ناخواسته ایجاد نشود؛ کاربر بعد از بازشدن ابزار روی «تولید» می‌زند.
- اتصال Render، Upload واقعی، صفحهٔ معرفی دو‌زبانه و راهنمای فنی حفظ شده‌اند.

نصب:
محتویات پوشهٔ docs بسته را در پوشهٔ docs مخزن جایگزین کنید.
