# یادداشت حریم خصوصی نسخهٔ عمومی آزمایشی

دانشیار برای آزمایش آموزشی طراحی شده است. کاربران نباید PDF محرمانه، اطلاعات دانشجو، پاسخ‌نامهٔ آزمون، یا محتوایی را که مجوز پردازش آن را ندارند بارگذاری کنند.

در این نسخه:

- فایل برای استخراج متن و تولید خروجی روی سرور موقتاً نگه‌داری می‌شود؛
- دسترسی هر کتاب با یک شناسه و توکن تصادفی مخصوص همان مرورگر محافظت می‌شود؛
- فایل و داده‌های وابسته پس از زمان تعیین‌شده حذف می‌شوند؛
- در صورت فعال بودن OpenAI، بخش‌های مرتبط کتاب برای پاسخ یا تولید محتوا به API ارسال می‌شوند؛
- این نسخه جایگزین سیاست رسمی حریم خصوصی دانشگاه یا قرارداد پردازش داده نیست.
