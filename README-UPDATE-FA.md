# به‌روزرسانی رابط دانشیار

این نسخه بخش «پیش‌نویس‌ها» را اضافه می‌کند.

## نصب

1. فایل `docs/index.html` موجود در مخزن را با فایل جدید جایگزین کنید.
2. پوشهٔ `.nojekyll` را حفظ کنید.
3. در GitHub Desktop بنویسید:

   Add persistent draft repository to public interface

4. Commit to main و سپس Push origin را بزنید.

## امکانات جدید

- ذخیرهٔ خروجی‌ها در LocalStorage مرورگر
- مشاهده و پیش‌نمایش کامل
- جست‌وجو و فیلتر
- تأیید پیش‌نویس
- دانلود به‌صورت فایل متنی
- حذف یک مورد یا پاک‌کردن کل مخزن

نکته: این ذخیره‌سازی مخصوص دموی استاتیک GitHub Pages است و داده‌ها در همان مرورگر نگه‌داری می‌شوند. نسخهٔ عملیاتی Backend باید پیش‌نویس‌ها را در پایگاه داده و حساب کاربری ذخیره کند.
