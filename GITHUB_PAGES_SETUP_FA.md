# راه‌اندازی GitHub Pages برای دانشیار

1. پوشهٔ `docs` را در ریشهٔ مخزن کپی کنید.
2. در GitHub Desktop تغییرات را Commit و Push کنید.
3. در مخزن GitHub وارد `Settings` شوید.
4. از منوی سمت چپ `Pages` را انتخاب کنید.
5. در بخش `Build and deployment` تنظیم کنید:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`
6. روی `Save` بزنید.
