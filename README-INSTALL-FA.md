# قابل مشاهده و قابل دانلود شدن همهٔ پیش‌نویس‌های دانشیار

این بسته موارد زیر را اضافه می‌کند:

- بازشدن خودکار پیش‌نویس بعد از تولید
- نمایش خوانای خلاصه
- نمایش ساختاریافتهٔ آزمون با سؤال، گزینه‌ها، پاسخ و توضیح
- نمایش کارت‌مانند فلش‌کارت‌ها
- نمایش درختی نقشهٔ مفهومی
- نمایش پیش‌نویس اسلاید و دانلود فایل واقعی PowerPoint
- دانلود خلاصه، آزمون، فلش‌کارت و نقشهٔ مفهومی با فرمت‌های:
  - Markdown
  - Text
  - JSON
- بازیابی و ساخت مجدد PowerPoint اگر فایل موقت Render پس از Restart حذف شده باشد
- ذخیرهٔ تمام خروجی‌ها در Backend و نمایش آن‌ها در بخش «پیش‌نویس‌ها»
- حفظ DOI رسمی 10.5281/zenodo.21481330

## نصب

محتویات این بسته را در ریشهٔ مخزن کپی کنید و فایل‌های قبلی را جایگزین کنید.

پیام Commit پیشنهادی:

Enable preview and download for every generated draft

سپس:

Commit to main
Push origin

Render و GitHub Pages پس از Push دوباره Deploy می‌شوند.

## آزمایش

1. صفحه را با Ctrl+F5 باز کنید.
2. یک PDF متنی بارگذاری کنید.
3. به «تولید محتوا» بروید.
4. یکی از خروجی‌ها را تولید کنید.
5. پیش‌نویس باید خودکار باز شود.
6. از پنجرهٔ پیش‌نمایش فرمت دانلود را انتخاب کنید.
7. برای اسلاید، فایل PPTX واقعی دانلود می‌شود.
